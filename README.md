# Video Annotation Tool

Мінімальний інструмент анотації відео прямокутниками на базі open source CVAT.

## Функціонал

- ✅ Завантаження відео (.mp4) та фото (jpg, png, gif, webp)
- ✅ Конвертація відео у фрейми через `<video>` + Canvas
- ✅ Завантаження зображень безпосередньо
- ✅ Навігація по фреймах (Previous/Next) - для відео
- ✅ Інструмент Rectangle:
  - Створення прямокутників drag & drop
  - Перетягування прямокутників
  - Зміна розміру (4 кути)
  - Видалення (Delete або правий клік)
- ✅ Експорт анотацій у JSON
- ✅ Локальне збереження (localStorage)

## Встановлення

```bash
cd video-annotation
npm install
```

## Запуск

```bash
npm run dev
```

Відкрийте браузер на адресі, яку покаже Vite (зазвичай http://localhost:5173).

## Використання

1. Натисніть "Завантажити відео/фото" та виберіть файл:
   - Для відео: .mp4, .webm, .mov та інші формати
   - Для фото: .jpg, .png, .gif, .webp та інші формати
2. Відео буде конвертовано у фрейми (1 fps), фото відображається одразу
3. Використовуйте мишу для створення прямокутників:
   - Лівий клік + перетягування - створення нового прямокутника
   - Лівий клік на прямокутнику + перетягування - переміщення
   - Лівий клік на куті прямокутника + перетягування - зміна розміру
   - Правий клік на прямокутнику - видалення
   - Кнопка Delete - видалення виділеного прямокутника
4. Натисніть "Експорт JSON" для збереження анотацій

## Формат експорту

```json
{
  "frames": {
    "0": [
      { "x": 100, "y": 120, "w": 200, "h": 150 }
    ],
    "1": [],
    "2": [
      { "x": 300, "y": 80, "w": 120, "h": 90 }
    ]
  }
}
```
